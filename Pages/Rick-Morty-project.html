Rick & Morty Project

Project description

This project is about a Vue-based website where you can create, modify, and delete tasks. It is themed around the University of Alabama's American football team, which I am passionate about.

On this website, you can register, interact with your tasks, and modify your player data in the profile section, such as height, player number, and position.

I invite you to explore the site and try out all its features. If you have any suggestions to improve the user experience, please let me know. I hope you enjoy using the site as much as I enjoyed creating it.
