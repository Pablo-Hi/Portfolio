Rick & Morty Project

Project description

The project is a Rick and Morty themed website that shows detailed information about all the characters in the series. The page is programmed using React as the framework for creating the user interface and is connected to a Rick and Morty API that provides detailed information about each character.

The page displays a complete list of all the characters in the series in a table. The table shows the character's name, its species, its status, indicating whether the character is alive or not, and the number of episodes in which it has appeared.
